# DrifterAI v1.3.0 Patch Notes

## Added
- Added `GEMINI_API_KEY` and `GEMINI_MODEL` env support.
- DrifterAI can use Gemini with Google Search grounding for broader web answers.
- Gemini is now preferred for chat replies when configured.
- Groq remains as fallback if Gemini is not configured or fails.
- Daily AI tips can now use Gemini if Groq is not configured.

## Version
- Bot version is now `v1.3.0`.
