# DrifterAI v1.5.0 Patch Notes

## Added
- Daily random web-intel post to one channel only.
- Supported daily intel channels: `sc-news`, `ship-talk`, `loadouts`, `trade-routes`, `mining`, `intel-drops`.
- Posted web links are remembered so DrifterAI does not repeat the same links.
- `#patch-notes` checks daily and posts only new Star Citizen patch notes.
- `#game-status` checks hourly and posts only when platform status changes.

## Changed
- Old hourly RSI news automation is disabled by default.
- Bot version is now `v1.5.0`.
