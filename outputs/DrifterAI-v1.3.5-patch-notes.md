# DrifterAI v1.3.5 Patch Notes

## Changed
- Added a Railway workaround for Gemini keys when new variable names do not attach.
- If `GROQ_API_KEY` contains a Google Gemini key, DrifterAI can use it for Gemini.
- If `GROQ_MODEL` contains a Gemini model name, DrifterAI can use it for Gemini.
- Bot version is now `v1.3.5`.
