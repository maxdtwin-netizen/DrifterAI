# DrifterAI v1.4.0 Patch Notes

## Changed
- Removed Gemini from the chat answer path.
- DrifterAI now uses Groq for replies and Brave/Tavily for web search.
- Added `BRAVE_SEARCH_API_KEY`, `TAVILY_API_KEY`, and `WEB_SEARCH_PROVIDER`.
- Default Groq model is now Llama 4 Scout.
- Bot version is now `v1.4.0`.
